ROS机器人运动控制

发表于 2023-07-12 分类于 ROS学习 本文字数： 1.9k 阅读时长 ≈ 3 分钟

ROS机器人运动控制

  • 矢量运动控制和旋转运动控制

    • 对速度进行量化

    • 选择机器人底盘在地盘投影的中点

    • 建立三维坐标系，食指指向X轴，中指指向y轴，拇指指向z轴。三个方向的矢量合成

    • 沿x轴旋转正方向滚转运动，y轴俯仰运动，z轴偏航运动

    • ROS中速度控制即是矢量运动和旋转运动的三轴分量

  • 流程控制

    • 速度控制节点—速度控制话题—机器人核心节点
    • 速度控制节点消息类型：geometry_msgs/Twist
    • 速度控制话题：/cmd_vel

机器人运动控制的C++实现

  • 框架实现思路

    • 构建一个新的软件包，vel_pkg

    • 在软件包中新建一个节点，节点名字叫做vel_node

    • 在节点中，向ROS大管家NodeHandle申请发布话题/cmd_vel,并拿到发布对象vel_pub

    • 构建一个geometry_msgs/Twist类型的消息包vel_msg,用来承载要发送的速度值

    • 开启一个while循环，不停的使用vel_pub对象发送速度消息包vel_msg

    打开vs code

    • 在创建的软件包内，src目录内创建节点代码文件vel_node.cpp

    对其文件进行编译

    • 打开软件包的CMakelist.txt文件
    • build章节中复制三条编译规则到文件末尾，取消注释
      • add_executable
      • add_dependencies
      • target_link_libraries
    • 以上三条都为编译规则开头名字

    ctrl+shift+b进行编译

    借助开源工程来完成机器人和环境的仿真
