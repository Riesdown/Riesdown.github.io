B站-ROS学习

ubuntu系统简单操作

  • 添加到收藏夹——类似windows固定到任务栏

  • win+空格键——输入法切换，shift键也可

  • CTRL+alt+t，终端打开

  • 简单命令

    • ls，简单查看当前所有目录内容
    • cd ..回到上一级目录
    • mkdir 创建新的目录
    • cd ~回到主文件夹
    • tab键，补全指令内容，且只有唯一内容
    • gedit 文本编辑器
    • source 执行文件中的指令序列 .sh文件
    • echo 显示的意思
    • .bashrc 环境变量的赋值
    • gedit ~/.bashrc 编辑主文件夹下的bashrc文件
    • ls-a 显示所有文件
    • sudo 以管理员权限执行这条指令

  • 在 ~/catkin_ws/src目录下执行git clone 后面添加GitHub项目网址

  • scripts 目录用于放置脚本文件和Python程序 安装依赖包

  • 安装工作空间内 的依赖包后，编译完成后，在工作空间文件夹下执行source指令

  • 使用source指令载入工作空间的环境设置

    • source ~/catkin_ws/devel/setup.bash

gedit ~/.bashrc

    • 加载的就是apt下载的那些软件包地址

  • source ~/catkin_ws/devel/setup.bash
    • 对应的是catkin工作空间里的软件包地址

  • 尽量选择系统中存在的软件包

源码编译

  • CmakeList.txt文件中 Build部分

    • 声明一个C++可执行文件
    • 为这个包添加一个可执行程序
    • 可执行文件与包的名字保持一个名字
    • CTRL + S保存
    • CTRL+shift +B进行编译

  • 运行节点

    • 运行ros roscore
    • 添加环境配置
    • rosrun

  • ctrl + shift +w
