八、小结

Navigator 主循环的核心机制：

  1. 以 nav_state 为中枢：Commander 通过 nav_state 驱动模式切换，Navigator 只执行选定的模式
  2. 生命周期统一调度：NavigatorMode::run() 保证切入/切出/后台的干净执行
  3. MissionBlock 语义统一：所有模式产生的任务最终通过同一翻译层转化，保证一致性
  4. 安全优先：围栏检查在模式执行之前，任何位置违规都会优先触发保护行为
