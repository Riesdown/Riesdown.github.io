前言

随着无人机上层应用的普及，需要用到ROS对无人机进行路径规划，导航，避障等的地方越来越多，简单记录下学习一些ROS的心得。

  • 虚拟机安装

    • 安装简单

    • 运行速度慢
    • 安全备份简单

  • 硬盘安装

    • 安装复杂
    • 硬件支持好
    • 运行速度快
    • 安全备份复杂

  • sudo 提升用户权限

基础操作（Linux）

  • 软件安装源（在软件和更新）
    • 家庭网络选择阿里云
    • 校园网edu

简单指令

  • pwd查看当前路径
  • cd ..跳到上级目录
  • mkdir 创建新的文件夹
  • ls 查看当前路径所有文件夹
  • touch 创建一个文件
  • mv 移动文件，相当于剪切作用
  • cp 拷贝路径/文件名 目标路径/重命名文件
  • rm 删除任意文件
  • rm -rf 删除文件夹
  • sudo 提升用户权限
    • sudo apt-get update
  • 指令 –help
    • 会弹出指令使用说明

C++/Python极简基础

  • sudo apt-get install g++

    • 安装g++编译器

  • sudo apt-get install python

    • 安装python编译器
  • g++ c++_for.cpp -o c++_for
//g++是编译器，c++_for.cpp是执行文件名字，-o c++_for是编译成可执行文件名字
./c++_for,编译可执行文件

ROS安装步骤

  • 添加ROS软件源

    • sudo sh -c 'echo'"deb http://packges.ros.org/ros/ubuntu $(lsb_release -sc)main">/etc/apt/sources.list.d/ros-latest.list'

安装ROS

  • sudo apt update
sudo apt install ros-melodic-desktop-full

设置环境变量

  • echo "source /opt/ros/melodic/setup.bash">>~/.bashrc
source~/.bashrc

启动ROS master

  • roscore

启动海龟控制节点

  • rosrun turtlesim turtle_teleop_key
